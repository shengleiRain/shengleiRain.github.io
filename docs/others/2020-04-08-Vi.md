## Vi的模式

- 末行模式（command-line mode）：操作文本文件，以及设置编辑环境，例如保存文件等。按`：`进入
- 插入模式（insert mode）：编辑文本的模式，按`i`等键进入
- 普通/命令模式（normal/command mode）:进来就是正常模式，用来浏览和修改文本内容，`<esc>`就能返回正常模式
- 可视模式（visual mode）:相当于高亮选取文本后的普通模式
  - 以行为单位进行选取的可视行模式，使用`V`键进入
  - 以块为单位进行选取的可视块模式，使用`ctrl+v`进入

## 普通/命令模式

### 指令重复

- 数字+指令：例如10dd表示执行10次dd
- **.**  : 重复上一个指令

### 标记某个地方

| 指令        | 作用                                  |
| ----------- | ------------------------------------- |
| m，例如ma   | 标记某个地方，ma表示将这个地方标记为a |
| \`，例如\`a | 回到标记处                            |

> 注意：也可与其他指令配合，例如d\`a就表示将光标与标记a之间的内容全部删除

### move指令

| 指令       | 作用                                       |
| ---------- | ------------------------------------------ |
| h（left）  | 向左移动                                   |
| l（right） | 向右移动                                   |
| j（down）  | 向上移动                                   |
| k（up）    | 向下移动                                   |
| w（word）  | 向前移动一个单词，定位在单词的第一个字母处 |
| b          | 向后移动一个单词，定位在单词的第一个字母处 |
| (和)       | 按句子移动                                 |
| {和}       | 按段落移动                                 |
| ^和$      | 分别移动到一行非空单词的最前面和最后面      |
| 0      | 移动到一行的最前面             |
| gg   | 移动到文档的第一行             |
| G      | 移到文档的最后一行             |

### 删除指令

| 指令 | 作用                     |
| ---- | ------------------------ |
| x    | 删除当前光标处的一个字符 |
| dd   | 删除一行                 |
| dw   | 删除一个单词             |

### 复制和粘贴指令

| 指令   | 作用                 |
| ------ | -------------------- |
| yy，yw | 复制一行或者一个单词 |
| p      | 粘贴                 |

### 移动屏幕

| 指令 | 作用                     |
| ---- | ------------------------ |
| z    | 将当前行移到屏幕的最上方 |
| z-   | 将当前行移到屏幕的最下方 |
| z.   | 将当前行移到屏幕的最中间 |

### 撤销和恢复

| 指令   | 作用               |
| ------ | ------------------ |
| u      | 撤销前一个操作     |
| ctrl+r | 恢复前一个撤销操作 |

### 查找中上一个匹配和下一个匹配

| 指令 | 作用       |
| ---- | ---------- |
| n    | 上一个匹配 |
| N    | 下一个匹配 |

## 末行模式

一般用来保存文本和退出，以及设置编辑环境。

> 永久改变vim的编辑环境配置，可以修改`/etc/vim/vimrc`或者`~/.vimrc`（对于当前用户有用）

| 指令      | 作用               |
| --------- | ------------------ |
| :set nu   | 文本中显示行号     |
| :set nonu | 文本中取消显示行号 |
